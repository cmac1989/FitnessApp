# Fitness Coaching App

A full-stack mobile fitness application built with **React Native**, **Laravel**, and **MySQL**, designed to connect trainers and clients through personalized programming, messaging, and progress tracking.

>  This project is actively under development.

---

## Overview

This app enables trainers to manage clients, assign workout programs, and track progress, while clients can follow structured plans, communicate with trainers, and complete check-ins.

The goal is to create a scalable platform for **online coaching**, with a focus on mobility, movement quality, and structured progression.

---

## Tech Stack

### Frontend
- React Native
- Async Storage (state persistence)

### Backend
- Laravel (REST API)
- MySQL (hosted database)

### Integrations (In Progress)
- Stripe (payments)
- Push Notifications

---

## Features

### Authentication & Onboarding
- Trainer & Client registration flows
- Role-based onboarding
- Secure authentication
- Forgot password flow
- Email support system

### AI Program Generator
- Automatically generates workout programs
- Designed to assist trainers in scaling programming

### Program Management
- Assign programs to clients
- Structured workout delivery
- Expandable for future progress tracking

### Messaging System
- Real-time-style messaging between trainer and client
- Enables ongoing coaching and support

### Check-In System
- Clients submit progress updates
- Trainers can monitor adherence and performance

### State Persistence
- Async Storage used to maintain user session and app state
